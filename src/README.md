# Git Magic Commit

## Install

1. Install `git-magic-commit` globally using npm:

```bash
$ npm install -g @yskoht/git-magic-commit
```

2. Set up your OpenAI API credentials as environment variables:

```
export OPENAI_API_KEY="your-api-key-here"
```

## Usage

```
$ git-magic-commit
```
